# hc-simulador

Simulador da Copa do Mundo pra jogar com os amigos.

## Lembretes pro dia do jogo

- **Peça pra cada pessoa escolher o país com calma antes de confirmar.** Hoje, depois que alguém confirma o nome e o país, não dá pra voltar atrás e trocar de ideia pela própria tela, então vale avisar todo mundo antes de começar: "escolhe direito, porque depois não muda".
